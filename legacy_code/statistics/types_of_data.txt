Major types of data:
- Numerical
- Categorical
- Ordinal

Numerical:
Represents quantifiable thing that we can measure 
- Discrete Data: Integer based (ex: Number of purchases)
- Continuous Data: Infinite range of possible values (ex: height of person)

Categorical:
No inherent mathematical meaning (ex: Gender, Race, Product Category)

Ordinal:
Mixture of Categorical and Numerical data
ex: Movie ratings on a 1-5 scale
